# SwiftGame

A framework for creating cross-platform games written in Swift.
Inspired by MonoGame/XNA it will initially be structured very similarly, but is expected to diverge and become more idiomatic Swift as it progresses.

## Supported Platforms

Initial support is for Apple platforms, with support for Linux and Windows to be added after the initial implementation is functional.

* Desktop Operating Systems
  * Mac OS X
* Mobile Devices
  * iPhone/iPad
* Other
  * tvOS
